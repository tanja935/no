第一話　姉弟

　眼前には果てしなく広がる美しい海、緩やかな海風が潮の香りを部屋へと運んでくる。大理石をふんだんに使った広々とした豪華な部屋の中心に美しい海が良く見えるようにベッドが配置されている。その上に絡み合う二つの人影。大きく柔らかな枕を背もたれにして頬を寄せ合って海を眺めている。



「カストル、海よ。美しいでしょう？」

「うん……エルナト姉さん……とても美しいね……」

「フフ……じゃあ、姉さんにご褒美をもらえるかしら？　カストル」

　カストルはたしか一年前までは普通の姉弟だったと振り返る。それ以前からたしかに姉は自分を溺愛していた。母がいないせいもあってなのか、あらゆる面倒を見てくれた。エルカリア帝国の嫡男として生まれ、次期皇帝となることが決まっているせいか周りの大臣や貴族達もカストルには甘かった。しかし生まれつきなのか、とことん甘やかされて育ったにも関わらず気が弱く他人に対して強く出ることが出来なかった。それを見越して強欲な貴族や商人達がカストルを利用しようと近づいてきた。そんな自分を周囲から守ってくれたのが姉達であった。

　「姉さん、気持ちいいですか？　ここが気持ちいいのですか？」



　今度はカストルがエルナトに問う。エルナトは閉じていた目をうっすらと開き虚ろな目で愛する弟の顔を恥ずかしそうに眺める。エルナトは唇と噛みしめて答えない。



「ねぇ、ここが気持ちいんですよね？　答えて、姉さん」



　カストルはイジめるようにエルナトに問い詰め、そして固く勃起した乳首を噛む。



「えぇ、カストル気持ちいいわ、気持ちいいの。お願いもっと、もっとして」



　エルナトは恥ずかしそうに顔を背け頬をさらに紅潮させながら答える。カストルは満足そうに言われるがままに乳房を揉み、乳首を舐めまわした。エルナトは胸を突出し体を反らせて押し寄せる快感の波に抵抗していた。カストルの右手が胸から離れ脇腹を撫でながら足の付け根へと這い、そしてついにエルナトの秘部へと到達する。

「姉さん、どうして欲しいのですか？」

「……」

「気持ち良くないのですか？　もう辞めてしまいましょうか？」

「いや！！　ダメ、ダメよカストル！！　お願い辞めないで！！」



エルナトは顔を上げカストルを見つめて哀願する。



「では、僕はどうすれば良いのでしょう？」

「……うぅ、カストルのいじわる……分かっているくせに……アンッ」



　カストルは股間に置いた指をさらに動かし催促する。



「もう、私の可愛い弟はなんて意地悪なのかしら……ハァハァ……お願い口で、いえ、舌でお願い……舐めて、舐めて欲しいの。ハァハァ……アンッ……お願いカストル」



　カストルは体をズラしてエルナトの秘部にキスをして、舌で突起した部分を刺激する。エルナトはビクンと体を目いっぱい反らし左の手首を口に押し当て声を抑える。



「まったく姉さんはいやらしいですね。弟にこんなところを舐めさせるなんて」

「ごめんなさい、カストル、ハァハァ……イヤラシい姉でごめんなさい……あぁ……」

「いいですよ。大好きな姉さんのためですから」

「あぁ、カストル、カストル……愛……してる……わ……」



　刺激を必死に耐えながら苦悶の表情でカストルに答える。そんな姉の姿を見てカストルも興奮を抑えきれず貪るように姉の秘部を舐め続ける。エルナトの秘部からはどんどんと蜜が溢れだしシーツまでも濡らしていく。エルナトは両手でシーツを掴み体をイヤラしくくねらせカストルの舌から逃れようとするが両腕でしっかりと下半身を抑え込まれて逃れることが出来なかった。くびれた腰を動かすたびに口から声が漏れ絶叫に近くなっていく。



「あぁ、カストル、気持ちいいわ、でも、でももうダメよ、ハァハァ、もうダメ、イってしまうわ！！」



　カストルはその言葉を聞いても意に返さず執拗に舌を動かし続ける。



「あぁ、カストル、ダメダメ、あぁ、ハァハァ、いや、ダメよ、あぁ、ひぐっ、ひぃ」



　シーツを掴んでいた両手をカストルの頭に置いて髪を掻きむしる。エルナトは両足を閉じようと試みるがカストルはそれを許さず更にエルナトのク○トリスを攻め続ける。必死に耐えるがエルナトの体を知り尽くしているカストルは、愛液まみれになっている秘部に指をすべり込ませて激しくかき回した。いきなり指が体の奥まで侵入したことに体は敏感に反応してしまい、エルナトは上半身を限界まで反らせ激しく喘ぎ、そしてついに絶叫と共に果てた。



「あぁ、ハァ、ひっ、ひっ、ふぐぅ……あ、あ、ハァハァ……」

　姉達。カストル、カストル・メア・セイリオス・フォン・レイファードには二人の姉がいる。一人はエルナト・リア・イングリット・フォン・レイファード。二つ年上で今年15歳になった長女。そしてもう一人。エルナトの双子の妹、次女のシャウラ・カリア・ミルドレッド・フォン・レイファード。
