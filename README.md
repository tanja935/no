第四話　騎士

　カストルと二人の姉は、汗と愛液にまみれた体を城の中にある広い浴室で流した。カストルは仕立てられたスーツをしっかりと着込み、美しい豪華なドレスに身を包んだエルナトとシャウラを伴って食堂へと向かった。食堂は黒を基調とし金と銀で彩られたシックで落ち着いた雰囲気で端には数人の楽士が並びしっとりとした音楽を奏でていた。席に着くと侍女達がワインを注ぎ、皿に乗せた豪華な食事が運ばれ並べられた。上座にカストルが座り右側にエルナト、左側にシャウラといういつも通りの席順だ。シャウラは血の滴るようなレアの肉を細かく切り分け上品に口元に運んでいく。そしてそばに控えていた執事服に身を包んだ青年の侍従長、ルータスに問いかけた。



「アトリアはいつこちらに到着しますか？」

「アトリア様は十日程前に帝都を出立なさりましたので、順調であれば三日後には到着なされるかと」

「姉さん、アトリアにこちらの統治させるのですか？」

「ええ。実際に取り仕切るのは政治部のラオと軍部のグリマールだけど崇拝の対象は必要ですからね。それに私達も帝都に戻らなければいけないし。まぁ、少しの間だけですよ」

「姉さん、アトリアだけでは少し不安ですね」

「そうね。シャウラあなたの手駒から数人回せるかしら？」

「ええ。姉様。大丈夫よ」

「では、私達はアトリアをこの地での統治者として発布後帝都に戻りましょう。そうね。十日後にこちらを出発します。その準備を」

「かしこまりました。エルナト様」



　侍従のルータスは無表情で恭しく頭を下げた。表情を顔に出すことをエルナトが好まないことを心得ていた。ハンサムだがどこか陰のある落ち着いた雰囲気を持つ不思議な青年であった。



「アトリアと会うのは三ヵ月ぶりですね。あの子の脅える顔を久しぶりに見れるかと思うとゾクゾクするわ」

「シャウラ、食事中に品がないわよ。でも、そうね。久しぶりに会えるのはとても楽しみだわ。ねぇカストル」

「ええ。姉さん。そうですね」

「それと例の計画はどうかしら？」

「順調よ。姉様。明日から動き始めます」

「そう。手抜かりなく丁寧に、慎重にお願いね。より楽しめるように」



　エルナトの青く氷のような瞳の奥で残酷な光が宿っていた。









*************





「その内通者っていうのは信用できるのか？　ジーマ」

「ああ、私の父の部下だった人のツテだから問題ないはずだ、ガクス」



　無骨な武人風の年配の男に女性が答えた。ガクスと呼ばれた男はいまいち納得できないような表情でジーマを見ていた。ガクスは筋骨隆々の大男で大剣を背負っていた。食入るような目つきでジーマを見つめるがジーマは臆しない。ジーマも鉄の胸当てを装備し腰に剣を差していた。燃えるような赤毛に緑色の瞳の美しい女性だったが、18歳になったジーマもガクス同様鍛え上げられた肉体で、肌は浅黒く日焼けし身長も180cmを優に超えていた。女性らしい雰囲気とは言えず、野性味に溢れ獰猛どうもうな猛獣を思わせた。



　王都から少し離れた郊外の町の潰れた酒場風の木造の建物の中で十数人の男女が、転がっていた木箱や椅子などに座り会話の行方を見守っていた。全員戦士のような武装をしており、鍛え抜かれた肉体をしていた。



「いまは少しでも多くの情報が欲しい。だが慎重にやらねば我々などあっという間に踏み潰されて終わりだ」

「分かってる。その内通者は父の部下だった人の血縁でエルカリア帝国に嫁いだのよ。そして侍女として召し上げられて、かなり上の人のお世話をしているそうだ」

「ふむ。慎重すぎても情報は得られんか……わかった会ってみよう。ただし少しでも怪しいところがあったら死んでもらうが文句ないな」

「ええ、もちろんだわ」



　ジーマも己の信念のためにどんなことでもする覚悟はすでに出来ていた。



「王や王女様達、それと騎士様は無事なんでしょうか……」



　木箱に猫の様にしゃがんで座っていた一人の小柄な少年が呟いた。その少年の頭には猫のような耳と腰には尻尾が生えていた。その尻尾は力なく不安そうに垂れ下がっていた。



　その時入口の扉が2回ノックされた。全員が振り向き剣に手を掛ける。そして扉のそばにいた中年の男がドア越しに話しかけた。



「今日は休業だ。明後日来てくれ」

「ルーシアです」



　と相手は小声でドア越しに答えた。中年の男性はその名前を聞き扉を開いた。全員から緊張が解け、扉から現れた人物に視線が集まった。



「ルーシア　王都の状況はどうだった？」

「はい……」



　被っていたフードを外し、力なくうなだれた15～6歳の華奢な少女は口ごもっていた。その可愛らしい顔は暗く悲しげで今にも泣きだしそうだった。



「王様とお妃様方、そして王女様方は……亡くなられました………。レファレント広場に惨むごい姿で晒されていました……」



　全員が息を飲み込んだ。場の空気は悲しみと怒りで満ち溢れ淀んでいく。ジーマは強く唇を噛み血が流れ出ていた。トリアスディア王国が誇った美しき姫君達は失われたのだ。正当な王家の象徴として、残党の貴族や騎士達を糾合し帝国に刃向うことを恐れたのだろう。



「王女様は全員か？　五人共全員か？」

「確認できたのは4人、です……フィン様、タリア様、エマ様……そして、う、うぐ、メ、メイア様まで……。ルマリアーデ様は確認できませんでした……でも、恐らく、もう既に……」



　ついにルーシアは感情が抑えきれず泣き出してしまった。崩れ落ちそうになったルーシアを中年の男が支える。ルーシアは姫様付の侍女で可愛がられていたのだ。



「なんということだ……メイア様までも殺すとは……」

「うぅ、広場に柵が作られ、その中に、全員全裸で……、惨たらしく、ご、拷問された傷跡も……ひっく……」

「おのれ、帝国皇帝ハダルめ。絶対にこの報いは受けさせてやる」

「3人の騎士様達は？　その中にはいなかったのだな？」

「え、あ、ぐすっ、はい。その場にはおられませんでした」

「そうか。ルマリアーデ様と共にまだ囚われているのかもしれん。なんとかお救い出来ぬものか……」

「内通者からよい情報があればよいのだが」

「その者といつ会える？」

「恐らく4、5日後だな」

「その者の情報次第で、救出に出れるかもしれん各自準備をしておいてくれ」

「城に潜入するとしたらやはり地下の下水道から、例の秘密通路を使うのだな」



　と訝しむようにジーマは言った。数百年も前に作られたという秘密の通路で上級騎士だけがその存在を教えられていた。実際に使用したことはなく場所のみを知っているだけだった。



「そうだ」

「しかしそんな通路本当に存在するのか？」

「ああ、俺が若い時に一度だけ見に行ったことがある。たしかにあった」

「ではその通路使うことを前提とした計画を立てよう」

「そういえばルーシア、王都の城にはいまハダルの娘と王子がいたな？」

「はい。王都で捕えられた貴族様方の公開処刑の場にシャウラ姫が同席してました」

「なぜ王子と王女が……」



　ガクスは思案に耽った。皇帝は帝都から出ていない。すべて部下任せだったのだろうか。王子についての情報はほとんどない。双子の王女については、この世のものとは思えないほどの美しさだと世界中に鳴り響いている。それ他に帝国内では侍女や奴隷には厳しくあたり何人も殺しているという噂なら聞いたことがある。が、あくまで噂であって信憑性はなかった。まだ子供の王子と双子が何かできるとは思えなかった。すると長年帝国の軍部を動かしてきた将軍と団長が全て行い王子たちはお飾りだったのかもしれない。後に皇帝になった時のために武功を与えることはどこの国でも行われていた。しかし何度も小競り合いを繰り返してきたが、あの将軍のドリュスや団長のゲイケンに六か月やそこらで国を落とせるような実力があったとは思えなかった。何かが腑に落ちない。見たこともない新しい大規模魔術や毒ガス、戦略戦術、この短期間で全てが制圧されたのだ。謎は深まるばかりだった。





　そしてガクスとジーマは頷きあい、他の戦士達も各々準備のために店から出て行った。







**************





　シャウラは、食事が終わるとバルコニーに出て夜風に当たっていた。眼下には真っ暗で引き込まれそうな海が広がっており、波の音が絶え間なく続き鼻孔が潮の香りで満たされる。シャウラにとっても海を見るのは人生で初めてであったのでカストル同様感慨深いものがあった。その海を見ながらシャウラは囁いた。



「レマ。数人選んでアトリアの護衛を」

（かしこまりました）



　どこからともなく若い女性の声が返事をした。その声は抑揚もなくただ淡々と静かだった。そしてシャウラは一枚の金貨を海に向かってピンと親指で弾いた。



「アトリアが我儘を言ったら金貨ソレを使いなさい」

（心得ました）



　シャウラが投げた金貨には魔術が施されており、魔力を通せば既にアトリアの体内に寄生させている魔蟲イビルバグを活性化させ意のままに操ることが出来た。エルナトの魔眼ソウルイーターは、死者すら操る事が出来るが、魔蟲は生きた者しか操ることが出来なかった。寄生し、共生する。しかし術者の意向であれは、宿主を殺すことも可能だった。



　そしてまた少しの間、海を眺め、満足したのかシャウラは何も言わずにエルナトとカストルのいる部屋へと戻って行った。三人で海を眺めながら倒錯した悦びに浸るために。





　次の日、エルナトはいつも通りゴージャスなドレスを着こみ椅子に拘束されているルマリアーデ王女と向き合っていた。その椅子の向かいの長椅子には正装したカストルとシャウラが座っていた。ルマリアーデ王女は一昨日から椅子にずっと拘束されたままのため疲れ切ってうなだれていた。先程侍女に命じ体を洗わせドレスを着させてからまた椅子に拘束したのだった。



「おはよう、ルマリアーデ。気分は如何かしら？」

「良いわけありません」

「あらそれは残念ね。今日はとっても良い日なのに」

「騎士達に何をする気なのですか？」



　ルマリアーデは疲労の限界のようで、うなだれたままエルナトを見ずに問う。



「そうね。どうしましょう？　うふ、うふふふ」



　冷酷で残酷な響きを含むエルナトの含み笑いが、ルマリアーデの全身を恐怖に震えさせる。冷や汗が流れ恐ろしさで気がふれてしまいそうだった。必死に体の震えを抑え込もうとするがガクガクと更に大きく震えてしまった。身に迫る恐怖とはこれほどまでに耐えがたいものかと実感していた。



　エルナトは膝を付き、ルマリアーデの手を美しい陶器で作られたような白い手で握った。



「震えているわ。怖くなんてないわ。大丈夫よ」



　エルナトは真っ青なルマリアーデの顔を覗き込む。そして目が合った。ルマリアーデは美しく残虐な笑顔を張り付けたエルナトの表情を凝視する。もう何も考えられなかった。怖くて怖くて発狂しそうだった。次々に父や母、そして姉妹たちの拷問された姿が思い浮かぶ。



「あ、あぁ……、はぁはぁ、あ……」



　もう喉がひりつき、声さえまともに出せなかった。そしてドレスのスカートが濡れ縛り付けられた椅子の足を伝って液体が流れ落ちた。



「あらあら。やはり姉妹ね。あなたも漏らしてしまったの？」



　もうルマリアーデは恐怖と羞恥で心が一杯だった。



「シャウラ」

「はい。姉様」



　シャウラは近くにあった小さい純金のベルを振って鳴らすと、隣に通じる部屋の扉が開き三人の拘束された女性が衛兵に引き出されてきた。



「「「姫様！！　」」」



　三人共うなだれたルマリアーデを見つけると悲壮な表情で叫んだ。鎧を着こみ鍛え上げられた肉体の三人はこの国では聖騎士とも呼ばれ国民から絶大な人気を誇っていた騎士達であった。



「お前ら姫様を離せ！！」



　14歳で騎士になり天才の名を欲しいままにした16歳のオウカ。華奢で身長も低いが剣技に優れトリアスディアで右に出るものはいなかった。まだあどけない顔立ちで可愛らしい。そして生意気そうな表情と赤茶色の瞳が子供っぽい印象を持たせていた。ツインテールの金髪を激しく揺らせながらエルナトに飛び掛かろうとしているが左右の腕を衛兵に掴まれ抑え込まれていた。



「元気がいいのね……。あなた、たしかオウカさんだったかしら？」

「やめろオウカ。こいつはエルナトだ」



　シェダルは拘束されたままエルナトに飛び掛かろうとしたオウカを止める。シェダルは美しい白く長い髪を持ったエルフだった。年齢は解らないが見た目は20歳そこそこの知的な美女だった。人とは違った顔立ちで瞳も灰色というより銀に近かった。顔立ちも整っているが、人よりもやや面長な印象だった。耳も合わせたように長く尖っていた。弓の名手でエルフ特有の視力と筋力で数々の武功を立てた戦士だった。



　そのやり取りの間、メラクは静かに部屋の配置や人数、ルマリアーデの状態等を観察していた。メラクは歴戦の勇者を思わせる重厚な印象の燃えるような赤毛の女性だった。ジーマの姉でもある彼女は、獰猛な猛獣を思わせる雰囲気を醸し出し、同時に戦の女神のような荘厳で勇ましい風格を持ち合わせていた。身長は高く筋肉質で185cmはありそうだった。弱冠18歳で父の跡を継ぎ、20歳で聖騎士の中でもトップに立たった。剣技や体術に優れ魔法抵抗が高い武人で軍略や戦術にも明るかった。そしてその美しい顔を厳しい表情に歪め、鋭い視線をエルナトに向けていた。



　オウカはエルナトの名前を聞いて睨みつけながら後じさる。



「私をご存じなのですか？　シェダルさん」

「ああ、容姿端麗で賢く、魔法魔術をその若さで自在に操る殺人狂のエルカリア帝国の姫君エルナト・リア・イングリット・フォン・レイファード姫。そしてそっちにいる全く同じ顔をしたのが妹のシャウラ姫だろう？」

「よく御存じなのね。エルフというのは情報ネットワークでもお持ちなのかしら？」

「それにしても聞いてたのと違うな。噂では感情の無い無口な姫君と聞いていたが」

「ふふっ、そんなことはありません。私達も楽しいことがあれば笑うのですよ。ふふっ」



　ビクッとルマリアーデが笑い声に反応し体を震わせる。そして涙と恐怖で一杯になった表情で騎士たちに向き直った。



「お前！！　姫様に何をした！！」



　オウカが声を荒げ叫んだ。



「あら、ルマリアーデにはまだ何もしていませんわ。ああ、暴れ無いように拘束はしてますけどね」

「王と他の姫君はどうした？」



　メラクの低い怒りに満ちた声が静かに部屋に響いた。



「ふふ、ルマリアーデ。どうなったかこの方々に教えて差し上げなさい」



　エルナトはメラクからルマリアーデへと視線を移した。ルマリアーデは恐怖と怒りが入り混じった瞳で懸命に恐怖と闘いながらエルナトを睨んだ。



　エルナトはそんなルマリアーデにゆっくりと近づき両手で頬を挟み視線を合わせた。そして微笑を浮かべる。



「強い瞳ね。いいわ。とても綺麗よ、ルマリアーデ」



　そう言うと、エルナトはゆっくりとルマリアーデに顔を近づけ唇を重ねた。ねっとりとしたエルナトの熱い舌がルマリアーデの口内に侵入し蹂躙する。



　その光景を唖然とした顔で三人の騎士は見守っていた。ルマリアーデは突然のこの行為と恐怖で、理解が追いつかず固まってしまっていた。
